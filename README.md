### Usage:
```
git clone https://github.com/pashayogi/UnfollowBot
cd UnfollowBot
chmod +x UnfollowBot
nano unfollow.txt (put your unfollow here)[untuk mengganti target unfollow]
./UnfollowBot.sh / bash UnfollowBot.sh
```
